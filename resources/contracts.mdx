<Note>
  Addresses below are **devnet** deployments. Mainnet addresses will be published before production launch.
</Note>

## Key on-chain accounts

| Account | Description |
|---------|-------------|
| **Market** | Per-market state: status, expiry, outcome mints, vault |
| **USDC vault** | Holds collateral for a market's positions |
| **Platform fee vault** | Accumulated platform overround |
| **Maker fee intake** | Per-maker claimable fees |
| **AMM pool** | YES/NO reserves and LS-LMSR state |
| **Parlay pool** | LP-backed parlay collateral |

## Verification

All trades, resolutions, and redemptions emit on-chain events indexed by **buukie-indexer**. You can verify any transaction on [Solana Explorer](https://explorer.solana.com) using your custodial wallet address or transaction signature.
